# Portfolio Project

> Portfolio project built while studying **Kotlin + Spring Boot**.

이 프로젝트는 **『코틀린과 스프링 부트로 웹 개발 시작하기』**를 기반으로 진행하는 실습 프로젝트입니다.

책의 예제를 단순히 따라 하는 것이 아니라, 학습한 내용을 정리하고 Java와 Kotlin의 차이점, 실무 적용 포인트를 함께 기록하는 것을 목표로 합니다.

---

# 🎯 Learning Goals

- Kotlin 문법 익히기

- Spring Boot 기반 웹 개발

- Spring Data JPA

- MVC & Layered Architecture

- Spring Security

- Docker를 이용한 배포
