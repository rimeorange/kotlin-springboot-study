# Kotlin + Spring Boot Study

Java Backend Developer로서 Kotlin을 실무에 적용하기 위해 진행하는 학습 저장소입니다.

이 저장소는 단순히 책 예제를 따라 하는 것이 아니라, Kotlin 백엔드 개발에 필요한 기술을 학습하고 직접 실습하며 정리하는 것을 목표로 합니다.

---

# 🎯 Goals

- Kotlin 언어 이해 및 실무 활용
- Spring Boot를 이용한 백엔드 개발
- JPA를 활용한 데이터 접근
- Spring Security 적용
- Coroutine 및 비동기 프로그래밍
- Docker를 이용한 애플리케이션 배포
- Kotlin 기반 백엔드 아키텍처 학습

# 📌 Commit Convention

| Type | Description |
|------|-------------|
| feat | New feature |
| fix | Bug fix |
| refactor | Code refactoring |
| docs | Documentation |
| test | Test code |
| chore | Build, dependency, configuration |

Example:

```text
feat: implement presentation controller
docs: add chapter01 study notes
refactor: simplify service layer
test: add repository tests
```

# 🚀 Purpose

This repository records my journey of learning Kotlin for backend development.

Rather than simply following tutorials, I focus on understanding concepts, comparing Kotlin with Java, and organizing practical notes that can be applied to real-world projects.
